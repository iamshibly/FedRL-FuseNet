# FedRL-FuseNet

FedRL-FuseNet is an explainable federated learning framework for brain tumor MRI classification.

This repository contains the core experiment notebooks for model development, preprocessing analysis, fusion ablation, federated learning evaluation, reinforcement-guided selection, baseline comparison, and external validation.

## Project Overview

The project investigates a federated and explainable deep learning pipeline for brain tumor MRI classification. The workflow includes adaptive contrast enhancement, fusion-based feature learning, federated optimization, reinforcement-guided configuration selection, baseline comparison, external validation, and explainability analysis.
